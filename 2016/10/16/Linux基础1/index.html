Linux基础(一)

Posted on 2016-10-16   |   In Linux   |     |   visitors

本篇博客简易介绍一下Linux入门的一些问题，对Linux有兴趣的朋友可以在
[鸟哥私房菜之Linux基础命令]上查看Linux的一些知识，巩固自己所学的

前言

Linux是一种自由和开放源代码的类UNIX操作系统，所以Linux就是一套操作系统，它提供了
了一个完整的操作系统当中最底层的硬件控制和资源管理的完整架构。

为什么要学习Linux?

Linux本身就是基于Unix概念开发而来，所以也继承了Unix系统稳定有效率的特点，Linux也是
基于GPL授权下的产物，所以使用者使用Linux时不需花费大量费用，Linux的安全性也是有保障
的，其漏洞也会及时修补，另外Linux可同时允许多任务多用户，Linux于用户来说，保密性能
比较好，不耗资愿等。

redhat7和redhat6的异同

操作系统安装(异)

rhel7是一站式安装
rhel6是实验环境搭建

网卡配置文件(同)

RHEL6/RHEL7:编辑/etc/sysconfig/network-scripts/ifcfg-eth0
1.静态网络配置

退出编辑，
rhel7:systemctl restart network
rhel6:/etc/init.d/network restart #重启服务(动态网络配置同此)
2.动态网络配置

修改IP地址

rhel6:setup 或 修改网卡配置文件
rhel7:nmtui 或 修改网卡配置文件
nmtui #文本中的网络设定工具
nm-connection-editor #图形中的网络设定工具
nmcli connection delete连接名称###删除此连接
nmcli connection add con-name westos ifname eth0 type ethernet autoconnect yes #添加动态网络
nmcli connection add con-name westos ifname eth0 type ethernet ip4 ip/24 #静态网络

主机名配置文件

rhel6:/etc/sysconfig/network
rhel7:/etc/hostnaq

hostname保存在/proc/sys/kernel/hostname,其值是系统启动时从rc.sysinit读取的
配置文件修改完成后，重启可显示结果或直接使用sysctl kernel.hostname命令
使其生效

服务启动和开机自启动

1.服务启动
rhel6: service 服务名 restart | reload | start | stop | status
rhel7: systemctl restart | start | reload | stop | status 服务名
2.开机自启动
rhel6:chkconfig –list 服务名

rhel7:systemctl is-enabled 服务名

防火墙服务名称

rhel6:iptables
rhel7:firewalled
